# CarTeam

A premium e-commerce website for car accessories, parts, and tires. Built with modern web technologies to provide a seamless shopping experience for automotive enthusiasts.

## Features

- **Product Categories**: Repuestos (Parts), Llantas y Rines (Tires), Interior, Exterior, Audio y Video, Car Care
- **Interactive Product Gallery**: Featured products with detailed modals
- **Shopping Cart**: Add items, view cart, proceed to checkout
- **Responsive Design**: Optimized for desktop and mobile devices
- **Multi-language Support**: Spanish and English
- **Dark/Light Theme Toggle**
- **WhatsApp Integration**: Direct contact for inquiries
- **Promotional Sidebar**: Flash offers and deals

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with CSS variables for theming
- **JavaScript**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Inter font family for typography

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through categories and products
4. Add items to cart and proceed to checkout
